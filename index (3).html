⚙️ ¿Qué es una Función?

Empecemos desde el principio. Sin miedo, esto es muy fácil de entender.

🍹 Analogía: La Licuadora Mágica
Imagina que tienes una licuadora mágica. Le metes frutas (entrada) y siempre te da un jugo diferente según la fruta (salida). ¡Eso es una función! Cada entrada tiene exactamente UNA salida.

📥

Variable Independiente (x)

Es lo que TÚ controlas. Es la entrada de la función. También llamada "dominio". Tú decides qué número entra.

Dominio
📤

Variable Dependiente (y)

Es el resultado que sale. Depende de lo que metiste. También llamada "rango". La función decide qué sale.

Rango
🔗

Notación f(x)

Se lee "f de x". Significa que f es la función y x es el valor de entrada. f(3) significa: evalúa la función cuando x = 3.

Notación
📌 ¿Cuándo algo NO es una función? ▶

Para que algo sea función, cada entrada debe tener exactamente UNA salida.


❌ NO es función: x = 1 da f(x) = 3 y también f(x) = 7. ¡Dos salidas para la misma entrada!

✅ SÍ es función: x = 1 siempre da f(x) = 5. Única y consistente.

🧪 Prueba de la línea vertical: En una gráfica, si trazas una línea vertical y corta la curva en MÁS DE UN PUNTO → No es función.

🔢 El Dominio

¿Qué números pueden entrar a la función? ¡Vamos a descubrirlo!

🎭 Piénsalo así: El dominio es como la lista de invitados a una fiesta. No todos pueden entrar. Algunos números están prohibidos porque causarían problemas matemáticos.

📋 ¿Qué provoca restricciones?

➗

Divisiones

No puedes dividir entre cero. Si el denominador puede ser 0 → ¡ese x está prohibido!

f(x) = 1/x → x ≠ 0
Dom: (-∞,0)∪(0,+∞)
√

Raíces Cuadradas

No puedes sacar raíz de números negativos (en los reales). El contenido debe ser ≥ 0.

f(x) = √x → x ≥ 0
Dom: [0, +∞)
log

Logaritmos

Solo se puede calcular el log de números positivos. Nunca de cero ni negativos.

f(x) = log(x) → x > 0
Dom: (0, +∞)

📝 Notación de Intervalos

Intervalo Abierto
(a, b)
No incluye los extremos. Como ir de a a b sin tocarlos.
Intervalo Cerrado
[a, b]
Incluye los extremos. a y b están adentro.
Semiabierto
[a, b) o (a, b]
Un extremo incluido y el otro no.
Hasta el infinito
(-∞, +∞)
Todos los reales. Sin restricciones.
Unión
A ∪ B
Todo lo de A más todo lo de B. Se usa para "huecos".
Desigualdad
x ≥ 3
x puede ser 3 o cualquier número mayor que 3.

🪜 Cómo encontrar el dominio — Paso a Paso

Función Racional: Tiene una fracción con x en el denominador.
1

Identifica el denominador

En f(x) = (x+2)/(x²-9), el denominador es x²-9

2

Iguala el denominador a cero

x² - 9 = 0 → x² = 9 → x = ±3

3

Excluye esos valores

x ≠ 3 y x ≠ -3 del dominio

4

Escribe el dominio

Dom f = (-∞,-3) ∪ (-3,3) ∪ (3,+∞)
Función Radical: Tiene una raíz cuadrada (o par). El contenido debe ser ≥ 0.
1

Identifica el radicando

En f(x) = √(2x - 6), el radicando es 2x - 6

2

Pon ≥ 0 y resuelve

2x - 6 ≥ 0 → 2x ≥ 6 → x ≥ 3

3

Escribe el dominio

Dom f = [3, +∞)
Función Logarítmica: El argumento del log debe ser estrictamente positivo (> 0).
1

Identifica el argumento del logaritmo

En f(x) = log(x² - 4), el argumento es x² - 4

2

Pon > 0 y resuelve

x² - 4 > 0 → (x-2)(x+2) > 0 → x < -2 o x > 2

3

Escribe el dominio

Dom f = (-∞,-2) ∪ (2,+∞)
Funciones Lineales y Polinomios: ¡No tienen restricciones! Todos los reales.
1

Verifica que no hay raíces, logs o denominadores con x

f(x) = 3x² + 5x - 7 → Solo potencias enteras positivas, sin fracciones con x ni raíces.

2

El dominio es todos los reales

Dom f = (-∞, +∞) = ℝ
⚠️ Errores comunes:
• Olvidarse de probar ±√(número) en racionales
• Poner > 0 en raíces en lugar de ≥ 0 (la raíz de 0 sí existe, es 0)
• No descomponer bien el denominador antes de resolver

📊 El Rango

¿Qué valores puede producir la función? ¡Es más fácil de lo que parece!

🎯 Piénsalo así: Si el dominio son los ingredientes que pueden entrar a tu cocina, el rango son todos los platos posibles que puedes preparar. ¿Qué platillos puedes hacer?
📉

Rango en Funciones Lineales

f(x) = mx + b: la línea va de -∞ a +∞. El rango es todos los reales (salvo que m=0).

f(x) = 3x + 1 → Ran = ℝ
⛰️

Rango en Parábolas

Depende del vértice y si abre hacia arriba (a>0) o abajo (a<0).

f(x)=x²+2: Vértice(0,2)
Abre ↑ → Ran = [2, +∞)
🌊

Rango en √x

La raíz cuadrada siempre da resultados ≥ 0. Nunca da negativos.

f(x) = √(x-1)
Ran = [0, +∞)
🌀

Rango en 1/x

La función hiperbólica puede dar cualquier valor excepto 0.

f(x) = 1/x
Ran = (-∞,0)∪(0,+∞)
📈

Rango en Exponenciales

La función e^x o aˣ siempre da valores positivos, nunca negativos.

f(x) = 2^x
Ran = (0, +∞)
🔊

Rango en Logaritmos

El logaritmo puede dar cualquier valor real, desde -∞ hasta +∞.

f(x) = log(x)
Ran = (-∞, +∞)

🪜 Cómo encontrar el Rango

1

Identifica el tipo de función

¿Es lineal? ¿Cuadrática? ¿Radical? Cada una tiene su "comportamiento natural".

2

Para cuadráticas: busca el vértice

Vértice x = -b/(2a). Sustituye en f(x) para hallar el valor mínimo o máximo.

3

Considera transformaciones verticales

Si hay un "+k" o "-k" afuera de la función, el rango se desplaza k unidades.

4

Usa la gráfica como confirmación

Los valores del eje Y que toca la gráfica forman el rango. Lo que NO toca, no está en el rango.

🧮 Ejemplo avanzado: Rango de función racional ▶

f(x) = (x+1)/(x-2)


Paso 1: Llama y = (x+1)/(x-2)

Paso 2: Despeja x: y(x-2) = x+1 → yx - 2y = x + 1 → yx - x = 2y + 1 → x(y-1) = 2y+1

Paso 3: x = (2y+1)/(y-1). Esto existe para todo y excepto y = 1.

Ran f = (-∞,1) ∪ (1,+∞) = ℝ - {1}
La asíntota horizontal y = 1 es el valor que el rango no puede alcanzar.
